# AI-Infused Gamified Learning Platform (Healthcare Training)

An AI-powered e-learning system that automatically converts structured healthcare training content into interactive courses, lessons, modules, and quiz assessments.

This project demonstrates the integration of **Generative AI**, **backend automation**, and **database-driven course generation** for modern digital learning platforms.

---
## 🎯 Use Case
Designed for:

- Healthcare compliance training
- Corporate learning platforms
- AI-driven instructional design
- Gamified learning systems

---
## 🚀 Project Overview

This system uses AI to transform raw training documents into:

- Structured learning modules
- Lesson summaries
- Automatically generated quiz questions
- Database-ready course content

The goal is to reduce manual instructional design effort and enable scalable gamified learning experiences.

---

## 🧠 Key Features

- Automatic course generation from text files
- AI-generated lesson summaries
- Quiz generation (MCQ, True/False, Scenario-based)
- Module classification using AI
- MySQL database integration
- Structured course export as JSON
